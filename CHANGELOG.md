# Changelog

## [0.1.0] - Early Access

First public early-access release of the Tessera MCP server.

### Added

- Read tools for architecture trees, elements, search, and file-to-element lookup.
- Write tools for creating and updating `architecture.md` files.
- Context tools for project rules and element context.
- Validation tools for staleness, orphan folders, broken links, mixed layers, and file drift.
- Diagram and docs tools for `.mermaid.md` sources and compiled SVG documentation.
- Scaffold tools for bringing existing codebases into Tessera.
- Protocol bootstrap tools for guided `.tessera-protocols/` initialization.
- Workflow tools for drift review, documentation update planning/application, PR summaries, project bootstrap, architecture improvement suggestions, Mermaid diagram generation, system maps, and release readiness checks.

### Notes

- This is an early-access release intended for dogfooding and feedback.
- Tool names and response shapes may still change before a stable v1.0.
